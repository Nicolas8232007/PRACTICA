# Task Manager

Proyecto de practica para construir una aplicacion full stack con buenas practicas.

## Tecnologias

- Frontend: React con Vite
- Backend: Java con Spring Boot
- Base de datos: PostgreSQL en Supabase
- ORM: Spring Data JPA / Hibernate
- Control de versiones: Git

## Objetivo

Construir una aplicacion de gestion de tareas con autenticacion, API REST, persistencia en base de datos y una interfaz web.
